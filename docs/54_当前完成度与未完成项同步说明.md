# 当前完成度与未完成项同步说明

## 1. 文档目的

这份文档用于给下一位执行型 AI 一个更直观的同步视图：

1. 这个项目现在到底完成了什么；
2. 还有什么没完成；
3. 哪些事项已经在本地与远端双端验证过；
4. 下一步最值得继续推进的是什么。

## 2. 项目目标（当前不变）

这是一个面向学生的 **AI 学习整理系统**，目标仍然是做成一个 **比赛级完整成品**，而不是企业级 SaaS。

固定方向仍然是：

- 本地 PostgreSQL：存结构化数据
- 本地文件存储：存上传原文件

不要改回 MinIO 优先、云对象存储优先或 SQLite 主存储。
