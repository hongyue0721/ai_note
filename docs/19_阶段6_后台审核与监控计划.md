# 阶段 6：后台审核与监控计划

## 1. 目标

开始补齐后台最关键的两块：

1. 审核任务能力
2. 基础监控能力

## 2. 本阶段最小交付

### 审核任务

1. `GET /v1/review/tasks`
2. `POST /v1/review/tasks/{task_id}/decision`

### 监控 / 概览

1. `GET /v1/dashboard`
2. `GET /v1/admin/monitor/overview`

## 3. 当前策略

比赛级阶段先满足：

- 管理员能看到待审核任务；
- 管理员能审批 / 拒绝 / 修正任务；
- 管理员能看到任务统计、审核统计、失败数、最近错误信息。

暂不做：

- 告警系统
- 历史趋势图后台持久化
- 完整内容管理后台

## 4. 验收标准

1. 管理员可查询审核任务列表；
2. 管理员可提交审核决策；
3. 仪表接口可返回关键统计；
4. 监控接口可返回服务/任务基础状态。
