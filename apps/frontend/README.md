# Frontend

frontend 由两个 Vue 3 + Vite 应用组成：`student` 面向用户工作台，`admin` 面向管理员观测与配置。

## student

- 通过 `space_key` 进入用户空间
- 工作台包含：上传笔记、询问问题、笔记管理、详情页、编辑页
- 支持文件拖拽上传、本地预览分类、确认入库
- 支持 AI 解题结果 Markdown 渲染、结果标签编辑、加入笔记
- 支持笔记详情编辑标签、删除、下载原文件、搜索与图谱概览

## admin

- 管理员登录单独一层
- 登录后分为两页：`数据管理` 和 `管理设置`
- 数据管理：状态与接口指标观察、用户笔记数量详情
- 管理设置：管理员用户名/密码修改、运行时文本模型/视觉模型配置

## 目录
